# mall-admin-web

## 项目简介

`mall-admin-web` 是一套电商后台管理系统的前端项目，基于 Vue + Element UI 实现。该项目为前后端分离架构，后端项目为 `mall-swarm` 微服务系统。系统主要包括商品管理、订单管理、会员管理、促销管理、运营管理、内容管理、统计报表、权限管理等功能模块。

### 1. 商品管理模块 (PMS)

#### 功能介绍

商品管理模块提供商品全生命周期的管理功能，包括商品信息维护、分类管理、品牌管理、属性管理、库存管理等。

| 子模块 | 功能说明 |
| --- | --- |
| 商品列表 | 商品的增删改查、上下架、批量操作、导出 |
| 商品分类 | 多级分类管理、分类排序、分类图标设置 |
| 品牌管理 | 品牌信息维护、品牌Logo管理、品牌推荐 |
| 属性管理 | 商品属性分类、规格属性、参数属性配置 |
| 库存管理 | SKU库存查询、库存预警、库存调整 |
| 报损管理 | 商品报损登记、报损审核、报损统计 |
| 回本分析 | 商品成本分析、利润计算、回本周期 |

### 搭建步骤

#### 1. 克隆项目

```bash
git clone <项目地址>
cd mall-admin-web
```

#### 2. 安装依赖

```bash
npm install
```

#### 3. 配置后端接口

修改 `config/dev.env.js` 文件中的 `BASE_API`：

```javascript
// 本地后端接口（通过网关访问）
BASE_API: '"http://localhost:8201/mall-admin"'

// 或使用在线接口
BASE_API: '"https://admin-api.macrozheng.com"'
```

#### 4. 启动开发服务器

```bash
npm run dev
```

#### 5. 访问应用

打开浏览器访问：http://localhost:8090

## 部署说明

### 构建生产版本

```bash
npm run build
```

构建完成后，`dist` 目录下的文件即为生产版本。

### Nginx 配置示例

```nginx
server {
    listen 80;
    server_name admin.example.com;
    
    location / {
        root /path/to/mall-admin-web/dist;
        index index.html;
        try_files $uri $uri/ /index.html;
    }
    
    location /api/ {
        proxy_pass http://localhost:8201/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```
